# Boot info
Add system data to ssh screen after ssh login

* Server name
* Server release
* CPU info
* Memory info

Note: Server release read from /etc/redhat-release file

# Workout
Script collect system info, them save to myinfo.txt, them put data from myinfo.txt into /etc/motd
